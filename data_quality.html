Méthode 1 : Distance de levenshtein¶

La distance de Levenshtein est une distance donnant une mesure de la diffrence entre deux chanes de caractres. Elle est gale au nombre minimal de caractres qu'il faut supprimer, insrer ou remplacer pour passer dune chane lautre.
L'ide est de calculer la distance entre deux lignes comme tant la somme des distances de chaque attribut des deux lignes pris deux deux, ie :

distance(l1,l2) = distance(l1['given_name'],l2['given_name']) + distance(l1['surname'],l2['surname']) + ...

Nous commenons d'abord par importer la fonction edit_distance qui permet de calculer la distance de levenshtein :

In [63]:
from nltk.metrics.distance import edit_distance

La fonction suivante calcule la distance entre deux littraux (attributs) :

  • Si les valeurs sont identiques, ou si l'une des valeurs est NaN, alors la distance est gale 0.
  • Si les types des deux attributs sont diffrents, alors la distance est infinie.
  • Sinon, renvoyer la distance de levenshtein (En castant les attributs en chaine de caractres).
In [64]:
def distance(val1,val2):
    if val1==val2 or pd.isna(val1) or pd.isna(val2):
        return 0
    elif type(val1)==type(val2):
        val1 = str(val1)
        val2 = str(val2)
        return edit_distance(val1,val2, substitution_cost=1, transpositions=False)
    else:
        return np.inf

La fonction suivante fait tout simplement la somme des distances des attributs de deux lignes pris deux deux :

In [65]:
def distance_row(row1,row2):
    return sum([distance(x,y) for x,y in zip(row1,row2)])

Et enfin, on implmente une fonction qui permet de calculer la distance entre une ligne donne en paramtre, avec toutes les autres lignes, en ne renvoyant que les indices des lignes pour lesquelles la distance est infrieure 10 (ie pas plus de 10 suppressions, transformations, additions dans tous les attributs afin de passer de la ligne i la ligne j ).

In [66]:
def distance_vector(df,i,max_dist=10):
    n = df.shape[0]
    dist_vec = []
    for j in range(n):
        if i==j:
            continue
        d = distance_row(df.loc[i],df.loc[j])
        if d<=max_dist:
            dist_vec.append(j)
    return pd.Series(dist_vec)

Méthode 2 : Score de similarité¶

L'ide est de rduire le temps de calcul de la distance en ralisant une comparaison simple entre les attributs, et en attribuant un score de similarit de 1/m si les deux attributs sont identiques.</br> En d'autre termes, un score de 0 veut dire que les deux lignes sont totalement diffrentes, un score de 1 veut dire que les lignes sont identiques.

Nous implmentons la fonction qui calcule le score de similarit entre deux lignes de la manire suivante :

similarit(l1,l2) = nb_attributs_identiques/nb_attributs

Puis nous rccuprons les lignes qui sont similaire avec un certain threshold (par dfaut 0.5), c'est dire que deux lignes seront considrs comme identiques si elles ont au moins 50% des attributs qui sont identiques.

NB : Deux attributs NaN ne sont pas considres comme identiques, cela nous convient car nous ne voulons pas avoir un haut score de similarit pour deux lignes avec beaucoup de valeurs manquantes. C'est pour cela aussi que nous choisissons un threshold assez bas.

In [71]:
def df_similarity(df,i,threshold=0.5):
    row_sim = (df.loc[i]==df).sum(axis=1)/df.shape[1]
    return list(row_sim[row_sim>threshold].drop([i],errors='ignore').index)

Notre fonction calcule pour chaque sample, la liste de ses doublons (grce la mthode df_similarity implemente ci-dessus), et supprime toutes les lignes les plus similaires (plus de 50% de similarit). Puis, retourne la dataframe sans ces doublons

def detect_duplicates(df):
    n = df.shape[0]
    for i in range(n):
        if i not in df.index:
            continue
        sims = df_similarity(df,i)
        if len(sims)>0:
            df.drop(sims,axis=0,inplace=True)
    return df
